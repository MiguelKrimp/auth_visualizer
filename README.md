# auth_visualizer

A starter TypeScript project for authentication visualization tooling.

## Scripts

- `npm run dev` - Run the app directly with TypeScript.
- `npm run build` - Compile TypeScript to `dist/`.
- `npm start` - Run compiled JavaScript from `dist/`.

## Project Structure

- `src/index.ts` - Entry point.
- `tsconfig.json` - TypeScript compiler settings.
